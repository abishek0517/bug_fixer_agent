Auto Bug Fixer Agent - Project Phases

Goal:
Build a Python-only agentic AI project that can detect bugs, ask a local LLM for fixes, test generated code, remember attempts, and improve over time.

Learning Rules:
- The student writes and updates the Python code
- Explain each new idea briefly and in simple words
- Build one small step at a time
- Show the complete revised file when code changes are needed
- Test after each meaningful step
